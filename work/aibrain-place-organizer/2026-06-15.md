## seed script構成
追加したコマンド:
- `npm run inspect:private-data`
- `npm run seed:places`

`npm run seed:places` の挙動:
- `data/private/` 配下のCSVをparse
- normalized itemに変換
- ファイル名からsource_list_nameを推定
- dedupe
- classification / region classification
- Supabaseにupsert
- source_linksを作成/更新
- import_batches / snapshots / snapshot_itemsを作成
- 前回snapshotと比較して差分を記録
- CSVから消えたsource_listはsource_links.active=false
- placesは物理削除しない

Supabase env未設定時の確認済みメッセージ:
`Supabase接続情報が未設定です。.env.local に NEXT_PUBLIC_SUPABASE_URL と SUPABASE_SERVICE_ROLE_KEY を設定してください。`
